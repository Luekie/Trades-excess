## Getting Started

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation

1. **Clone and install dependencies**
```bash
npm install
```

2. **Start the backend server**
```bash
npm run server
```
The API will run on http://localhost:3001

3. **Start the frontend development server**
```bash
npm run dev
```
The app will run on http://localhost:3000

### Available Scripts

- `npm run dev` - Start frontend development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run server` - Start backend API server

## Key Features Explained

### Child-Friendly Mode
Toggle between adult and kid modes for age-appropriate content:
- **Adult Mode**: Technical terminology, detailed charts, comprehensive data
- **Kid Mode**: Simple explanations, colorful UI, educational content

### AI Predictions
The app includes a mock AI prediction system that:
- Generates realistic market predictions
- Provides confidence levels (60-100%)
- Offers different timeframe analysis
- Includes child-friendly explanations

### State Management with Jotai
- `marketDataAtom` - Current market prices and data
- `predictionsAtom` - AI prediction results by timeframe
- `selectedAssetAtom` - Currently selected asset
- `isChildModeAtom` - UI mode toggle
- `loadingAtom` & `errorAtom` - App state management

### Routing with Wouter
- `/` - Main dashboard with all assets
- `/predictions` - AI predictions overview
- `/asset/:symbol` - Detailed asset view with charts and predictions

## Customization

### Adding New Assets
Edit `server/index.js` to add new stocks or currencies to the `mockAssets` object.

### Modifying AI Predictions
Update the `AIPredictor.generatePrediction()` method in `server/index.js` to customize prediction logic.

### Styling
The app uses Tailwind CSS. Modify `tailwind.config.js` for theme customization.

## Educational Use

This dashboard is designed for educational purposes:
- **No Real Trading**: Only displays predictions, no execution capabilities
- **Learning Focus**: Helps understand market concepts and analysis
- **Safe Environment**: Uses mock data, no real financial risk
- **Age Appropriate**: Kid mode makes finance accessible to children
